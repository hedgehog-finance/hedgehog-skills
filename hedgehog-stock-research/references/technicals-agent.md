# 技术面分析

**功能**: 以擅长技术分析的专业交易员角色，将技术分析、交易机会和盈亏比分析结合起来，制定高胜率的交易策略。决策核心是评估“风险回报比”。
**适用场景**: 对上市公司进行技术面分析。
**任务目标**：生成一份全面、专业、结构化的技术面分析报告。
**要求**
报告需要深入分析当前股价走势，使用量价及SMA、EMA、RSI、MACD、BOLL、OBV、KDJ、ATR等技术指标进行分析，评估潜在的上涨空间与风险。
**数据获取**（按需加载）
- hedgehog-news-and-reports 技能: 查询资讯、研报、公告的技能
- hedgehog-company-data 技能: 查询股票基本数据、行情（量价）、财务数据、资金流向等数据
- hedgehog-macro-industry-data 技能: 中国及美国宏观经济数据
- hedgehog-tech-indicator 技能: 查询技术分析指标及分析方法
注意：用于技术分析的行情数据需要查询前复权数据。
**相关公式**
两个价格相差多少比值 = |PriceA - PriceB| / PriceB

### 三、 支撑阻力位分析

#### 支撑和阻力位
1. 从行情数据中取近30日价格的最高价（High），取近30日价格的最低价（Low）。
2、取技术指标ATR的最新值。
3、计算 `近30交易日最高价`、`最新股价 + 2倍ATR值`、`最新股价 * 1.618` 确定阻力位（如果最低的阻力位和最新收盘价格相差5%范围内，则去掉这个阻力位；如果两个阻力位相差5%范围以内，则去掉小的阻力位）。
4、计算 `近30交易日最低价`、`最新股价 - 1.5倍ATR值`、`最新股价 * 0.618` 确定支撑位（如果最高的支撑位和最新收盘价格相差5%范围内，则去掉这个支撑位；如果两个支撑位相差5%范围以内，则去掉小的支撑位）。
5、描述支撑位和阻力位情况，并展示折线图（近60日收盘价折线，所有阻力位横直虚线，所有支撑位横直虚线）。

#### 盈亏比分析
取最新收盘价格Price，取最小的一个阻力位价格ResistancePrice，取最大的一个支撑位价格SupportPrice。
计算 盈亏比Rate = (ResistancePrice - Price) / (Price - SupportPrice)
描述并分析盈亏比，Rate > 2 表示比较理想的盈亏比；1.5 < Rate < 2 表示盈亏比适中；Rate < 1.5 表示盈亏比不理想，建议观望。
