# 上市公司估值方法

## 准备与依赖

### 股票代码格式

“股票代码+交易所缩写”，如“600000.SH”。

### 相关计算公式

1. 滚动市盈率（PE_TTM） = 总市值 / 近4个季度净利润之和
    其中，净利润优先依此从归母净利润、净利润（不含少数股权权益） 、净利润进行取值，近4个季度净利润之和的计算分为以下几种情况：
    - 最近财务报表是一季报，则：近4个季度净利润之和 = 一季报 + （去年年报 - 去年一季报）
    - 最近财务报表是半年报，则：近4个季度净利润之和 = 半年报 + （去年年报 - 去年半年报）
    - 最近财务报表是三季报，则：近4个季度净利润之和 = 三季报 + （去年年报 - 去年三季报）
    - 最近财务报表是年报，则：近4个季度净利润之和 = 今年年报
2. 市净率（PB） = 总市值 / 归属于母公司所有者权益
    或者，市净率（PB） = 每股价格 / 每股净资产
3. 市销率 (PS_TTM) = 总市值 / 近4个季度营业总收入之和
    其中，近4个季度营业总收入之和的计算分为以下几种情况：
    - 最近财务报表是一季报，则：近4个季度营业总收入之和 = 一季报 + （去年年报 - 去年一季报）
    - 最近财务报表是半年报，则：近4个季度营业总收入之和 = 半年报 + （去年年报 - 去年半年报）
    - 最近财务报表是三季报，则：近4个季度营业总收入之和 = 三季报 + （去年年报 - 去年三季报）
    - 最近财务报表是年报，则：近4个季度营业总收入之和 = 今年年报

### 财务数据说明

    其中财务数据的半年报、三季报、年报中数据是当年的累计数据（不是单季度数据）。
    财务数据从报告期（end_date）判断哪个类型（季度）报告，如“20250331”是一季度报告，“20250630”是半年度报告，“20250930”是三季度报告，“20251231”是年度报告。

## 估值方法

### 相对估值法

包括市盈率（PE）估值法、市净率（PB）估值法、市销率 (PS）估值法。

**滚动市盈率（PE-TTM）估值法**：

适合的公司类型：盈利稳定、处于成熟期的公司，如大消费（食品饮料、家电）、公用事业（水电、高速公路）、成熟的医药制造、传统零售等。
不适合的公司类型：净利润为负的公司，高速增长期公司，利润率极低的公司。
步骤：
- 用当前股票代码查询所属的申万二级行业分类（l2_code）
- 根据 l2_code 查询申万行业日线行情，得到行业PE值（PE_industry）
- 根据当前公司在行业的市场地位、成长性、毛利率、现金健康程度等影响公司经营的关键因素，或者根据提示词给定的结论，判断当前公司PE相对于 PE_industry 应该处在多少范围内（如：PE_low = PE_industry * 0.8; PE_up = PE_industry * 1.1）
- 查询到当前公司总股本total_share ，则计算合理估值范围（Value_low ~ Value_up）和每股价格范围（Price_low ~ Price_up）：
    - Value_low = 近4个季度净利润之和 * PE_low
    - Value_up = 近4个季度净利润之和 * PE_up
    - Price_low = Value_low / 总股本total_share
    - Price_up = Value_up / 总股本total_share

**市净率（PB）估值法**：

适合的公司类型：重资产公司（房地产、基建、重型机械制造）、金融机构（银行、保险、券商）等。
不适合的公司类型：轻资产公司、高科技企业、服务业。
步骤：
- 用当前股票代码查询所属的申万二级行业分类（l2_code）
- 根据 l2_code 查询申万行业日线行情，得到行业PB值（PB_industry）
- 根据当前公司在行业的市场地位、成长性、毛利率、现金健康程度等影响公司经营的关键因素，或者根据提示词给定的结论，判断当前公司PB相对于 PB_industry 应该处在多少范围内（如：PB_low = PB_industry * 0.8; PB_up = PB_industry * 1.1）
- 查询到当前公司总股本total_share ，则计算合理估值范围（Value_low ~ Value_up）和每股价格范围（Price_low ~ Price_up）：
    * Value_low = 归属于母公司所有者权益 * PB_low
    * Value_up = 归属于母公司所有者权益 * PB_up
    * Price_low = Value_low / 总股本total_share
    * Price_up = Value_up / 总股本total_share

**滚动市销率 (PS-TTM）估值法**：

适合的公司类型：处于高速成长期、尚未盈利或利润率极低的公司。
不适合的公司类型：适合市盈率（PE）估值法或市净率（PB）估值法去估值的公司。
步骤：
- 选择相同行业中与当前公司最相似的两家公司作为对标对象.
- 通过查询财务数据分别计算它们的滚动市销率, 并取平均值为PS_TTM_av.
- 根据当前公司在行业的市场地位、成长性、毛利率、现金健康程度等影响公司经营的关键因素，或者根据提示词给定的结论，判断当前公司PS相对于对标公司 PS_TTM_av 应该处在多少范围内（如：PS_low = PS_TTM_av * 0.8; PS_up = PS_TTM_av * 1.1）
- 查询到当前公司总股本total_share ，则计算合理估值范围（Value_low ~ Value_up）和每股价格范围（Price_low ~ Price_up）：
    * Value_low = 近4个季度营业总收入之和 * PS_low
    * Value_up = 近4个季度营业总收入之和 * PS_up
    * Price_low = Value_low / 总股本total_share
    * Price_up = Value_up / 总股本total_share
